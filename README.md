
# My Finance Tracker 💰

## 📌 Overview

This project is a simple and interactive Finance Dashboard built using React and Tailwind CSS. It helps users track their income, expenses, and understand their financial activity through visual insights.

---

## 🚀 Features

* 📊 Dashboard overview with Balance, Income, and Expenses
* 📈 Time-based (Line Chart) and category-based (Pie Chart) visualizations
* 📋 Transactions table with search functionality
* 👤 Role-based UI (Viewer/Admin)
* 💡 Insights section (Total, Highest Spending, Analysis)
* 🌙 Dark Mode toggle
* 📭 Empty state handling for better user experience

---

## 🛠️ Tech Stack

* React
* Tailwind CSS
* Recharts

---

## 📂 Project Structure

* Modular components for clean and reusable code
* Separate folders for components and data
* Simple and maintainable state management using React hooks

---

## ▶️ How to Run

```bash
npm install
npm run dev
```
